# F1 Tyre Stats Card

Tyre compound data, stint history, and current tyre age.

## Features

- Current tyre compound (SOFT/MEDIUM/HARD/INTERMEDIATE/WET)
- Tyre age (laps)
- Stint history per driver
- Team color bars
- Compound color coding

## Data Sources

- f1.0.tyres.stints (JSON)
- f1.0.tyres.current (JSON)
- f1.0.standings.drivers (JSON)
